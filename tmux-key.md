C-o                           向前旋转当前窗口中的窗格。

C-z                           暂停tmux客户端。

!                             将当前窗格打破窗外。

"                             将当前窗格一分为二，顶部和底部。

#                             列出所有粘贴缓冲区。

$                             重命名当前会话。

%                             将当前窗格分成左右两部分。

&                             杀死当前窗口。

'                             提示选择窗口索引。

(                             将附加的客户端切换到上一个会话。

)                             将附加的客户端切换到下一个会话。

,                             重命名当前窗口。

-                             删除最近复制的文本缓冲区。

.                             提示索引以移动当前窗口。

0~9                           选择窗口 0 到 9。

:                             输入tmux命令提示符。

;                             移至先前活动的窗格。

=                             从列表中选择要以交互方式粘贴的缓冲区。

?                             列出所有键绑定。

D                             选择要分离的客户端。

L                             将附加的客户端切换回上一个会话。

[                             进入复制模式以复制文本或查看历史记录。

]                             粘贴最近复制的文本缓冲区。

c                             创建一个新窗口。

d                             分离当前客户端。

f                             提示在打开的窗口中搜索文本。

i                             显示有关当前窗口的一些信息。

l                             移动到先前选择的窗口。

m                             标记当前窗格（请参阅select-pane -m）。

M                             清除标记的窗格。

n                             切换到下一个窗口。

○                             在当前窗口中选择下一个窗格。

p                             切换到上一个窗口。

q                             简要显示窗格索引。

r                             强制重绘附加的客户端。

s                             以交互方式为附加的客户端选择一个新会话。

t                             显示时间。

w                             以交互方式选择当前窗口。

x                             杀死当前窗格。

z                             切换当前窗格的缩放状态。

{                             将当前窗格与前一个窗格交换。

}                             将当前窗格与下一个窗格交换。

~                             显示以前的消息tmux，如果有的话。

PageUp                        进入复印模式并向上滚动一页。

Up，Down，Left，Right         切换到当前窗格上方、下方、左侧或右侧的窗格。

M-1~M-5                       以五种预设布局之一排列窗格：偶数水平、偶数垂直、主水平、主垂直或平铺。

Space                         在下一个预设布局中排列当前窗口。

M-n                           移动到带有铃铛或活动标记的下一个窗口。

M-o                           向后旋转当前窗口中的窗格。

M-p                           移动到带有铃铛或活动标记的上一个窗口。

C-up，C-Down，C-Left，C-Right 以一个单元格为单位调整当前窗格的大小。

M-UP，M-DOWN，M-LEFT，M-RIGHT 以五个单元格为单位调整当前窗格的大小。
